# AugmentedAnalyst

A collection of AI-powered tools for the modern CTI practitioner — built to work across Claude, Gemini, and local models (LM Studio). Each component takes raw threat intelligence as input and produces structured, analyst-ready deliverables without requiring prompt engineering expertise from the end user.

---

## What's In This Repo

### `claude-skill/` — CTI Fusion Analyst (Claude)

A Claude skill that ingests a raw threat report and produces a full suite of structured CTI deliverables in a single session. Designed for use at claude.ai (paste the skill prompt), Claude for Teams/Enterprise (upload to skill library), or via the Claude API (use as a system prompt).

**Deliverables produced:**

| Deliverable | Audience |
|---|---|
| Core Analysis | CTI analyst — ATT&CK mapping, IOC table, review gate |
| Threat Assessment | CTI team and operational leadership |
| Executive Brief | Senior leadership — no jargon, one page |
| Risk Mitigation Plan | All teams — owner-assigned action items |
| Threat Hunt Hypothesis | Hunt and IR teams |
| Red Team Emulation Plan | Offensive and purple teams |
| Patch Notification | IT and system owners — CVE-triggered |

See [`claude-skill/README.md`](claude-skill/README.md) for setup and usage instructions.

---

### `gemini-gem/` — CTI Fusion Analyst (Gemini)

The same CTI Fusion Analyst workflow deployed as a custom Gem in Google Gemini Advanced. Intended for practitioners working in Google Workspace environments or who prefer Gemini as their primary AI assistant.

See [`gemini-gem/README.md`](gemini-gem/README.md) for Gem setup and notes on Gemini vs. Claude behavior differences.

---

### `lm-studio-presets/` — CTI Preset Library (Local / Offline)

Seven focused system prompt presets for LM Studio. Each preset handles one deliverable from a raw source document. No internet connection required. Designed for air-gapped or offline environments and tested on Gemma 3n E4B (Q4_K_M) on a CPU-only machine with 16 GB RAM.

**Presets included:**

| Preset | Output |
|---|---|
| CTI - 1 Core Analysis | ATT&CK mapping, IOC table |
| CTI - 2 Detection Rules | Sigma rules |
| CTI - 3 Threat Assessment | Threat picture for CTI and leadership |
| CTI - 4 Executive Brief | Under one page, no jargon |
| CTI - 5 Risk Mitigation | Owner-assigned remediation actions |
| CTI - 6 Threat Hunt | Hunt procedures for IR teams |
| CTI - 7 Patch Notification | CVE remediation notice for IT/system owners |

See [`lm-studio-presets/README.md`](lm-studio-presets/README.md) for configuration, model compatibility, and preset installation instructions.
